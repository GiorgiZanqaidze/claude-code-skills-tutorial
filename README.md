# Claude Code Skills Tutorial

A hands-on tutorial for mastering **Claude Code** — Anthropic's CLI-based AI coding assistant.

## What is Claude Code?

Claude Code is an agentic coding tool that runs in your terminal. It can read/write files, run commands, search codebases, and help you build software faster — all through natural language.

## Prerequisites

- Node.js 18+
- A terminal (bash, zsh, PowerShell)
- An Anthropic API key or Claude Pro/Max subscription

## Quick Start

```bash
# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Start Claude Code
claude

# Or run a one-shot command
claude -p "explain this project"
```
